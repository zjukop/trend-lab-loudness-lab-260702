# Loudness Lab

A tiny open-source desktop app that makes every audio file the same perceived volume and generates beautiful before/after proof reports. It rides the visible demand for simple Python utilities around audio normalization/equalization, but differentiates with creator presets, LUFS targets, privacy-first local processing, and highly shareable visual reports.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Podcasters, YouTubers, musicians, audiobook narrators, lecturers, video editors, and anyone cleaning up mixed-volume audio folders.** and focuses on the niche: **Creator-friendly local audio cleanup tool combining batch normalization, simple equalizer presets, and visual reports for podcasters, YouTubers, musicians, and students.**.

## Key Features

- Drag-and-drop batch audio loudness normalization to streaming/podcast targets like -14 LUFS, -16 LUFS, and EBU R128
- One-click before/after loudness report with waveform thumbnails, peak/RMS/LUFS table, and shareable HTML export
- Built-in lightweight system audio presets: podcast voice, night mode, music cleanup, gaming clarity, and car playback
- Cross-platform desktop app packaged with Python + FFmpeg, with no cloud upload and privacy-first local processing
- Watch-folder mode that automatically normalizes new recordings, lectures, voice notes, or music folders
